# Assets Folder

This folder contains local images and GIFs for the Cluck N Smash website.

## Folder Structure

- **`gifs/`** - Place your GIF files here for carousels and hero sections
  - `hero-background.gif` - Main hero section background
  - `carousel-1.gif` - First carousel slide (Burger)
  - `carousel-2.gif` - Second carousel slide (Chicken)
  - `carousel-3.gif` - Third carousel slide (Burger)

- **`images/`** - Place your static images here
  - Menu item images
  - Location images
  - Other static assets

## How to Use Local GIFs

1. Download the GIFs from the links above
2. Rename them to match the expected filenames (carousel-1.gif, etc.)
3. Place them in the `src/assets/gifs/` folder
4. The website will automatically use your local files!

## Current Setup

The website is configured to use local GIFs from this folder first.
If a local GIF is not found, it will automatically fall back to the online GIFs from the links above.
