# Hotel Website

## Overview
This project is a static website for a hotel, designed to showcase its offerings in a visually appealing and user-friendly manner. The website is built using HTML, CSS, and JavaScript, incorporating modern styles and animations.

## Project Structure
```
hotel-website
├── src
│   ├── index.html          # Main HTML document
│   ├── css
│   │   ├── style.css       # Main styles for the website
│   │   └── animations.css   # CSS animations for enhanced user experience
│   ├── js
│   │   └── script.js       # JavaScript for interactivity
│   └── assets
│       └── fonts           # Custom font files
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Features
- Responsive design for optimal viewing on various devices
- Smooth animations for a modern user experience
- Interactive elements for user engagement
- Custom fonts for unique typography

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd hotel-website
   ```
3. Install dependencies (if any):
   ```
   npm install
   ```
4. Open `src/index.html` in a web browser to view the website.

## Usage
- Explore the hotel offerings displayed on the main page.
- Enjoy the animations and interactive features designed to enhance user experience.
